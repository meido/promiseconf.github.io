INSTRUCTIONS TO PREPARE THE CAMERA-READY COPY:

Full (short) papers must not exceed 10 pages, including references.

Authors will receive the rights management form directly from ACM shortly. Please make sure that you set email SPAM settings to allow emails from "rightsreview@acm.org".

The format of your paper must strictly adhere to the ACM SIG Proceedings Format. You can use the ACM Word or LaTeX style templates to ensure that. Templates are available at http://www.acm.org/sigs/publications/proceedings-templates.

Make sure you correctly classified your paper using the ACM Computing Classification categories and terms. ACM requires the three small sections 'Categories and Subject Descriptors', 'General Terms', and 'Keywords' to follow the abstract. The keywords can be freely chosen, but the content of the other two must be picked from the lists provided athttp://www.acm.org/about/class/1998

Articles must be in PDF format.

All fonts must be embedded in the PDF file. The following link contains instructions on that: http://www.acm.org/publications/WORD%20to%20PDF%20instructions.txt

All fonts must be Type 1 (scalable). Bit-mapped font types (like Type 3) are not acceptable.

All figures should be embedded in vector graphics or bitmaps with a resolution of at least 300 dpi.

You must include the rights management text and bibliographic strip sent by ACM in the camera-ready, on the bottom of the first page left column.

In order to include the correct rights management text and bibliographic strip in latex, authors will need to add the following commands immediately after \begin{document} in their .tex file: \conferenceinfo{PROMISE}{'14, September 17, 2014, Turin, Italy} \CopyrightYear{2014} \crdata{978-1-4503-2898-2/14/09}

In addition to that, ACM's style file may need to be manually modified in order to display the correct information. If you are using "sig-alternate.cls", please see attached the instructions on how to modify the style. Note that you need to complete the changes indicated below using the DOI and the right management text provided by the ACM.
